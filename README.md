# WeatherAPI-Project
WeatherAPI Project for XivTech

This is Jayesh Goyal from MSRIT-ISE 4th year. It is an assignment submitted with respect to Backend Development Intern. 

# Tech stack

- NodeJS
- Express
- HTML & CSS
- Jquery
- Bootstrap

# How to start server ?

- Install Dependencies : 
``` npm install ```

- Run the server :
``` node server.js ```
